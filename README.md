# The Turning
<i> Text Based Horror Game using LLMs </i>

<b> To survive, you must distinguish  Man and Machine </b>

## Features

- Human-written lines and AI-generated lines
- Characters will use a different set of lines every playthrough (Re-playable 2-3 times)
- At the end of the game, you will see the results of the choices you made
- Text to speech dialogue to truly immerse you in the world (Tentative)


## DEVELOPMENT: Git and Unity

- To install to local repo, run `git clone https://github.com/Akila-Fernando/ENSF-400-Grp11-TheTurning.git`
- To opne the project:
1. Open unity hub. 
2. Navigate to library and click add. Select "from disk"
3. Navigate to your local repo and select the `TheTurning` folder.
4. Follow any steps from unity hub. 
- To work on a new feature, use `git checkout -b feature_name`
- Regularly stage changes with `git add .` and commit them with `git commit -m "Message"`
- When your changes are mostly ready, push to your branch's origin. DO NOT PUSH TO MAIN OR I WILL EAT YOUR COMPUTER!!!!.
- Open a pull request and wait for approval. Once approved, your changes will be merged to head.
