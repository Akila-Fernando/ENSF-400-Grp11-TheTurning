# The Turning
<i> Text Based Horror Game using LLMs </i>

<b> To survive, you must distinguish  Man and Machine </b>

## Features

- Human-written lines and AI-generated lines
- Characters will use a different set of lines every playthrough (Re-playable 2-3 times)
- At the end of the game, you will see the results of the choices you made
- Text to speech dialogue to truly immerse you in the world (Tentative)
